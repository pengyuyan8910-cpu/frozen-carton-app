# 新增门店自动严格排柜入口

1. 下载并编辑 `新增门店配置模板.xlsx`，每行填写一类冰柜配置。
2. 在 GitHub 的 `data/new-store` 文件夹中直接上传这份表。文件名可以自定义，例如 `新增门店配置_和县生活馆.xlsx`；无需删除或覆盖旧文件。
3. 在 Actions 中等待“新增门店严格排柜草稿”显示绿色对勾。
4. 在 `data/drafts/` 下载结果工作簿、JSON 和复核报告。草稿不会覆盖正式底表。

## 严格约束

- 复用现有小程序的10%触发、外储容量和柜段占宽口径。
- 新店立柜柜4第1-4层只用作其他品类预留；第5层可排冻品；第6层是存储位。
- 除上述预留与存储位外，可用冻品柜段不允许空置。
- 柜段不超宽，雪糕冰品只进冰淇淋柜，建议外储不超配置上限（默认754L）。
- 四级品类优先集中，同SKU同柜型内集中陈列。
