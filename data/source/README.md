# 底表自动同步入口

以后只维护这里的当前版底表即可：

- `整箱到店数据测算_当前版.xlsx`：Excel底表入口，适合你手动调改表格后上传。
- `整箱到店数据测算_当前版.json`：小程序回传入口，适合你在小程序里新增门店、全店上新或调整陈列后导出再上传。

上传到 GitHub 后会自动执行：

1. 读取当前版底表。
2. 转换成小程序数据 `data/app-data.json`。
3. 生成复核报告 `data/verify-report.json`。
4. 生成版本信息 `data/version.json`。
5. 复核通过后自动提交数据并更新公共网页。

复核失败时不会更新 `data/app-data.json`，公共网页继续使用上一版已通过数据。

注意：如果同时存在 JSON 和 XLSX，系统会优先读取最近一次提交更新的文件；本地没有 Git 信息时，按文件修改时间判断。
