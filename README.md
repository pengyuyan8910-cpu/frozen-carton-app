# 冻品整箱到店统一小程序

10%触发口径 · 门店可执行 · 运营可调试

## 功能

- **总览看板**：门店外储需求排行、关键指标一览
- **门店执行**：逐店商品陈列执行明细
- **商品明细**：全量商品多维度筛选查询
- **周转风险**：外储周转风险评估
- **产品池管理**（运营）：产品池增删改、批量导入
- **补货测算**（运营）：当前库存下最多可补箱数测算
- **排柜调整**（运营）：商品陈列柜位灵活调整
- **柜段余量**（运营）：各柜段空间利用率
- **空位建议**（运营）：淘汰品释放空间的再分配方案
- **新增门店测算**（运营）：按冰柜配置严格测算新店方案
- **陈列图**：可视化门店陈列图，支持整层拖动交换
- **导入导出**（运营）：方案 JSON 导入导出

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。
